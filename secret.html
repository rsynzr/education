Rancangan Proyek: Web Remote Desktop Pribadi

Proyek ini bertujuan membangun sistem remote laptop pribadi yang bisa diakses dari browser dengan domain khusus, aman lewat HTTPS, dan tanpa port forwarding.

1. Nama Proyek

Remote Laptop Web

Tujuan utama: mengakses laptop Ubuntu dari mana saja melalui browser, tanpa perlu menginstal aplikasi tambahan di perangkat klien.

2. Tujuan dan Target Pengalaman

  • Buka browser.
  • Masuk ke website.
  • Login.
  • Laptop desktop langsung muncul.
  • Bisa mengontrol keyboard dan mouse.
  • Mendukung transfer file.
  • Aman melalui HTTPS.

3. Persyaratan Sistem

Host (Laptop Ubuntu Desktop LTS):

  • Docker dan Docker Compose
  • Apache Guacamole
  • guacd
  • MariaDB
  • XRDP
  • Cloudflare Tunnel

Client: browser modern di Android, iPhone, Windows, Linux, atau macOS.

4. Domain dan Jalur Koneksi

Domain yang dipakai: remote.rsynzr.cloud.

Internet
  ↓
Cloudflare DNS
  ↓
Cloudflare Tunnel
  ↓
Laptop Ubuntu
  ├── Docker
  ├── Apache Guacamole
  ├── guacd
  ├── MariaDB
  ├── XRDP
  └── Desktop Ubuntu

5. Fitur Minimum dan Tambahan

Fitur minimum:

  • Login dan logout
  • Dashboard
  • Remote desktop
  • Fullscreen
  • Clipboard
  • Copy paste
  • Upload dan download file
  • HTTPS
  • Session timeout
  • Multi-user

Fitur tambahan:

  • CPU usage
  • RAM usage
  • Disk usage
  • Network speed
  • Uptime
  • Temperatur CPU jika tersedia
  • Jumlah proses

6. Arsitektur Aplikasi

Backend: Node.js + Express.js

Frontend: React + Tailwind CSS + Vite

Remote Desktop: Apache Guacamole

Database: MariaDB

Deployment: Docker Compose

Reverse Proxy: Nginx

Tunnel: Cloudflare Tunnel

8. API yang Direncanakan

  • POST /login
  • POST /logout
  • GET /me
  • GET /system
  • GET /uptime
  • GET /network
  • GET /guacamole
  • GET /status
  • POST /wake

9. Dashboard dan Modul UI

  • Sidebar: Dashboard, Remote Desktop, File Manager, Settings, Logs
  • Home: CPU, RAM, Disk, Internet, Uptime, Status
  • Remote Desktop: fullscreen, clipboard, upload, download, keyboard mapping, mouse capture, reconnect otomatis
  • File Manager: upload, download, rename, delete, buat folder, drag and drop

10. Keamanan

  • HTTPS wajib
  • CSP Header
  • HSTS
  • Rate Limiting
  • Fail2Ban
  • Cloudflare Access opsional
  • Password kuat dengan hash
  • 2FA opsional
  • Tak ada port RDP terbuka ke internet
