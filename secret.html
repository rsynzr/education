Halaman Rahasia: Virtual Machine Ubuntu lewat Cloudflare Tunnel

Halaman ini berisi kerangka dan langkah-langkah instalasi untuk menyiapkan mesin Ubuntu, memasang Cloudflared, dan membuka akses layanan dengan aman.

1. Persiapan Virtual Machine Ubuntu

Awali dengan Ubuntu 22.04 atau 24.04. Anda dapat memilih instalasi lokal di VirtualBox, Multipass, atau menggunakan server cloud.

sudo apt update
sudo apt upgrade -y
sudo apt install -y curl git unzip

Rekomendasi VM lokal: 2 CPU, 4GB RAM, 30GB disk.

2. Membuat VM Ubuntu dengan Multipass

Multipass memudahkan pembuatan VM Ubuntu tanpa konfigurasi rumit.

sudo snap install multipass --classic
multipass launch 24.04 --name edu-ubuntu --mem 4G --disk 30G
multipass shell edu-ubuntu

Dalam VM, jalankan:

sudo apt update && sudo apt upgrade -y

3. Instalasi Cloudflared

Cloudflare Tunnel memungkinkan koneksi aman dari internet ke layanan lokal tanpa membuka port publik.

curl -L https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb -o cloudflared.deb
sudo dpkg -i cloudflared.deb
sudo rm cloudflared.deb

Login ke Cloudflare:

cloudflared tunnel login

Perintah ini membuka browser untuk autentikasi dan mengaitkan tunnel dengan akun Cloudflare Anda.

4. Membuat Tunnel untuk HTTP

Buat tunnel dan konfigurasi layanan web lokal yang berjalan di port 8080.

cloudflared tunnel create edu-tunnel
cloudflared tunnel route dns edu-tunnel edu.example.com
mkdir -p ~/.cloudflared
cat > ~/.cloudflared/config.yml <<'EOF'
ingress:
  - hostname: edu.example.com
    service: http://localhost:8080
  - service: http_status:404
EOF
cloudflared tunnel run edu-tunnel

Ganti edu.example.com dengan nama host di domain Cloudflare Anda.

5. Membuka SSH melalui Tunnel

Gunakan Cloudflared untuk mengamankan akses SSH tanpa membuka port 22 secara publik.

cat > ~/.cloudflared/config.yml <<'EOF'
ingress:
  - hostname: ssh.edu.example.com
    service: ssh://localhost:22
  - service: http_status:404
EOF
cloudflared tunnel run edu-tunnel

Siapkan klien SSH dengan host yang sesuai dan gunakan kunci SSH untuk autentikasi.

6. Setup Contoh Layanan Web

Jalankan layanan web sederhana di port 8080 untuk diuji melalui Tunnel.

sudo apt install -y python3
cat > ~/hello.py <<'EOF'
from http.server import HTTPServer, SimpleHTTPRequestHandler
HTTPServer(('0.0.0.0', 8080), SimpleHTTPRequestHandler).serve_forever()
EOF
python3 ~/hello.py

Setelah berjalan, akses layanan dari internet melalui https://edu.example.com.

7. Aturan Keamanan

  • Aktifkan UFW: sudo ufw enable dan batasi port yang dapat diakses secara lokal.
  • Gunakan Cloudflare Access untuk otentikasi tambahan jika diperlukan.
  • Berikan izin file konfigurasi Cloudflared yang benar: chmod 600 ~/.cloudflared/config.yml.

8. Instalasi tambahan untuk Ubuntu

Berikut paket penting lain untuk lingkungan Ubuntu yang lengkap.

sudo apt install -y build-essential net-tools htop tree

Untuk kerja projek, instal Git dan editor ringan:

sudo apt install -y git vim

Halaman rahasia ini hanya untuk eksplorasi konfigurasi Ubuntu dan Cloudflare Tunnel.
